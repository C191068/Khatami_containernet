the file which is oened by giving the first command we ate the below 


```

[Unit]
Description=Slice that limits docker resources 
Before=slices.target
[Slice]
CPUAccounting=true
CPUQuota=50%    
MemoryAccounting=true
MemoryHigh=2G  
MemoryMax=3G 
MemoryMaxSwap=10G

```
